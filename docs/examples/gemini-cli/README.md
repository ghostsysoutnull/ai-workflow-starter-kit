# Gemini CLI Examples

These examples are explicitly Gemini CLI-native.

They show:

- interactive usage
- file-aware prompting with `@`
- shell execution with `!`
- memory management with `GEMINI.md`
- session controls such as `/chat save`, `/compress`, and `/resume`
- headless output for automation

## Included examples

- [research-session.md](research-session.md)
- [implementation-session.md](implementation-session.md)
- [headless-automation.md](headless-automation.md)
- [full-worked-example.md](full-worked-example.md)

## How to use these examples

Do not copy them blindly.

Replace:

- directories
- prompts
- validation commands
- model selection
- output expectations

Keep the structure. The structure is the point.
