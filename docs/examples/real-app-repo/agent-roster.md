# Agent Roster

## Research Agent

Use for discovery, codebase mapping, and defect investigation.

Expected outputs:
- findings
- relevant files
- open questions
- recommended next step

## Planner Agent

Use for phased delivery plans, risk mapping, and sequencing.

Expected outputs:
- scoped plan
- assumptions
- validation checkpoints
- risk notes

## Implementer Agent

Use for focused code or doc changes.

Expected outputs:
- change summary
- touched files
- validation status
- residual risks

## Reviewer Agent

Use for acceptance review, risk review, and completeness checking.

Expected outputs:
- blocking issues
- notable risks
- optional improvements
- approval status
