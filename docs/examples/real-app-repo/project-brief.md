# Project Brief

## Project name

Acme Tasks

## Product summary

Acme Tasks helps teams track work across projects with shared task boards, notifications, and light reporting.

## Current priorities

- improve onboarding speed for new users
- reduce notification noise
- strengthen test reliability in the API layer
- improve documentation for contributors

## Delivery constraints

- preserve API compatibility where possible
- keep migrations reversible
- prefer incremental releases over large launches
- document operational impact for background job changes

## Success signals

- features ship with clear validation evidence
- onboarding docs match the real setup flow
- regression risk is visible before merge
- reusable prompts and playbooks are captured after successful work
