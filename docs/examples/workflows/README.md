# Example Workflows

These examples show how a Gemini CLI-driven project can run from request to result using the roles and templates from this starter kit.

## What each workflow includes

- scenario
- recommended role sequence
- sample task brief
- example prompts
- expected outputs
- validation steps
- common failure modes

## How to adapt

Replace:

- product names
- file paths
- constraints
- validation commands
- approval rules

Keep:

- the role sequence
- the thin-slice mindset
- the requirement to validate before handoff
