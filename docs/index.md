# Gemini-CLI Software Project Starter Kit

This documentation set is organized into reusable sections:

- [Agents](agents/README.md) for role definitions and handoff patterns
- [Best Practices](best-practices/README.md) for guardrails and quality guidance
- [Configuration Guidance](configs/README.md) for conventions and setup patterns
- [Prompt Examples](prompts/README.md) for reusable prompt structures
- [Gemini CLI](gemini-cli/README.md) for tool-specific operating guidance
- [Strategy Guides](strategies/README.md) for end-to-end delivery workflows
- [Templates](templates/README.md) for briefs, tasks, and handoffs
- [Examples](examples/README.md) for worked examples and sample repo packs
