# Documentation Generation Prompts

## Prompt: create a practical guide

> Write a markdown guide for this project task. Target a new contributor, keep the guide action-oriented, and include prerequisites, step-by-step instructions, common mistakes, and related links.

## Prompt: convert working knowledge into docs

> Turn the successful workflow from this task into a reusable markdown document. Include when to use it, required inputs, expected outputs, validation steps, and pitfalls.

## Prompt: improve an existing doc

> Review this markdown document for clarity, structure, and actionability. Improve headings, remove ambiguity, and add missing examples or next-step guidance without bloating the page.
